IP Address Intersection

Looks for IPs/Ranges that are only in set A, set B, or are in both A/B.

Syntax:

Single IPs, one or multiple per line
Ranges, hypen-separated (with or without spaces), one or multiple per line
CIDR ranges (one or multiple per line)
Ranges or IPs can be space or comma separated

Note: this runs entirely in-browser. No data is saved or transmitted.
